# Data Immersion & Wrangling Project

## Objective
The objective of this project is to clean, transform, and prepare retail datasets for analysis.

## Datasets Used
- sales.csv
- stores.csv
- features.csv

## Tasks Performed
- Loaded datasets using Pandas
- Explored dataset structure
- Checked missing values
- Checked duplicate rows
- Converted date columns to datetime format
- Performed outlier detection
- Created new features (year, month, day)
- Handled missing values in holiday_name column
- Merged datasets into one master dataset
- Generated final cleaned dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- VS Code

## Final Output
A fully cleaned and analysis-ready retail sales dataset.
